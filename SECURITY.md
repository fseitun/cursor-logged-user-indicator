# Security

If you believe you have found a security issue in this extension, please report it responsibly.

**Preferred:** use [GitHub Security Advisories](https://github.com/fseitun/cursor-logged-user-indicator/security/advisories/new) for the repository so the maintainers can assess and coordinate a fix before public disclosure.

This project reads local Cursor storage only and does not send account data over the network; still, reports about unexpected data exposure, unsafe subprocess usage, or supply-chain concerns are welcome.
